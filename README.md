# 🗃️ EstoqueSimples

Um sistema simples de controle de estoque desenvolvido em **C# Windows Forms**.  
Permite cadastrar produtos, visualizar em uma grade (DataGridView) e aplicar alertas visuais quando o estoque está baixo.

---

## 🚀 Funcionalidades

- Cadastro de produtos com **nome** e **quantidade**.
- Exibição dos produtos em uma **tabela dinâmica** (DataGridView).
- Alerta visual:
  - Linhas em **vermelho** e texto em **branco** quando a quantidade < 5.
  - Linhas em cores padrão quando a quantidade ≥ 5.
- Interface amigável com campos de texto e botões.
